# Template — copy this layout

This folder shows how **your** submission folder should look.

When you submit, make a folder with **your name** inside the correct week folder, like this:

```
submissions/
  week-1/
    aziz-karimov/          <- your name (small letters, dash instead of space)
      lesson-1.ipynb       <- your finished notebook
      lesson-2.ipynb
      lesson-3.ipynb
```

## Checklist before you submit
- [ ] Folder name is your real name, small letters, with a dash: `firstname-lastname`.
- [ ] All 3 notebooks for the week are inside.
- [ ] Each notebook runs with no error (Runtime → Run all).
- [ ] Your name is written in the first cell of each notebook.
- [ ] The assignment at the bottom of each notebook is done.

Do **not** put your files directly in `week-1/`. Always put them inside **your own name folder**.
