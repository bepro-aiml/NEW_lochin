# Fast-Track Module — Modules 1 to 4 (Catch-Up Course)

A 3-week catch-up course for new students. It covers the key ideas of **Modules 1, 2, 3, and 4** — enough to follow the class and pass the midterm.

**English level:** A2 (simple). **Background needed:** none. **Tool:** Google Colab.

---

## How each lesson works

Every lesson has **two files**:

1. **`theory.pdf`** — Read this **first**. It explains each idea in simple words: what it is, an example, **why** we use it, and the **common mistake** to avoid.
2. **`practice.ipynb`** — Open it in Google Colab. **Run every cell** and watch the result. At the bottom there is an **assignment**.

> The rule: **read the theory → run the practice → do the assignment.** Reading + doing = understanding.

### How to open a practice notebook in Colab
1. Go to https://colab.research.google.com
2. File → Upload notebook → choose the `practice.ipynb` file.
3. Run each cell with **Shift + Enter**.

## Tips for new students
- Do not rush. It is better to understand 3 lessons well than to skim 9.
- Lessons 2 and 3 (Python + pandas) are the hardest at the start. Give them the most time.
- If you are stuck for more than 15 minutes, ask your mentor or your study buddy.
- You do **not** have to memorize code. You must **understand** what each line does and **why** we use each tool — that is what the midterm checks.

## Tips for the mentor
- Pace: about 1.5–2 hours per lesson, 3 lessons per week.
- The assignment in each notebook = "run it, then comment every line in your own words" + short questions. This builds understanding without the friction of typing from a blank page.
- Pair each new student with a strong existing student.
- Use the mock test as a weekly check to see who is keeping up.
